# 🎲 Lottery Game

A modern and responsive **Lottery Game** built with **React.js** and **Vite**. The application generates random lottery numbers, tracks wins and losses, stores game history, and provides a clean, user-friendly interface.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/sunilpatel557/Lottery-game.git
```

Go to the project folder:

```bash
cd Lottery-game
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
